# Elevator System - Keyboard Controls

## Test Keys

Press any key to test the elevator system:

| Key   | Action                         |
| ----- | ------------------------------ |
| **A** | Go to Floor 1 (UP direction)   |
| **B** | Go to Floor 5 (DOWN direction) |
| **C** | Go to Floor 3 (UP direction)   |
| **D** | Go to Floor 2 (DOWN direction) |
| **E** | Go to Floor 4 (UP direction)   |

---

## Examples

**Test 1:** Queue Processing

1. Press **A** → Elevator goes to floor 1
2. While moving, press **C** → Elevator will go to floor 3 after floor 1

**Test 2:** Direction Preference

1. Press **C** (Floor 3, UP)
2. Press **D** (Floor 2, DOWN)
3. Elevator visits floor 3 first, then floor 2

**Test 3:** Same Floor

1. Elevator is at floor 1
2. Press **A** (Floor 1)
3. Doors will open and close (no movement needed)
